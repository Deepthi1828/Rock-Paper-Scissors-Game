# 🎮 Rock Paper Scissors Game

A modern and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer with real-time score tracking and an attractive user interface.

## 🚀 Features

- User vs Computer Gameplay
- Random Computer Choice Generation
- Real-Time Score Tracking
- Win / Lose / Draw Detection
- Interactive User Interface
- Responsive Design
- Smooth Hover Effects
- Modern Gaming-Themed UI
- Reset Game Functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Game Rules

- Rock defeats Scissors
- Scissors defeats Paper
- Paper defeats Rock
- If both choices are the same, the result is a Draw
